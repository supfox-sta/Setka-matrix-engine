# Сетка Matrix

Проект «Сетка Matrix» разворачивает собственную инстанцию Matrix на домене `setka-matrix.ru`. Сервер построен на базе открытого Synapse и предназначен для безопасного пространства общения внутри вашей локальной сети/сообщества.

## Что внутри
- **Домен:** `setka-matrix.ru`
- **Платформа:** Synapse (Matrix homeserver)
- **Цель:** собственная федеративная инстанция Matrix с контролем над инфраструктурой и персональными настройками модерации.

## Развитие проекта
1. Работайте в feature-ветках, применяйте `git pull` перед началом, чтобы быть в синхронизации.
2. Добавляйте осмысленные коммиты и описания (например, «Добавил новую комнату»).
3. После проверки отправляйте ветки в GitHub и создавайте pull request.
